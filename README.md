# plugin-terra-basic

## Installation 
1. **Clone this repo**
2. **Install the current nodejs LTS version from (https://nodejs.org/en/)**
3. **Run npm install within this project locally**
    
## Run developer mode 

1. **Start project**
    1. npm start
2. **Open plugin in your Browser**
    1. http://localhost:3002/
    
## Compile 

1. **Build plugin**
    1. npm run build
